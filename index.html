PRE-TRIP NOTES · 2026

Documents護照 / 文件

  • 護照效期至少 6 個月（2026/12/09 後皆需換發）
  • 申根區免簽，180 天內最多停留 90 天
  • 國際駕照 IDP — 南法 D9–D12 租車要用
  • 申根旅遊保險（建議含醫療轉送）
  • 機票、訂房、租車紙本備份各印一份

Currency貨幣 / 支付

  • 匈牙利使用 HUF Forint，不在歐元區
  • 法國 / 奧地利使用 EUR
  • 建議帶兩張不同網絡信用卡（Visa + Mastercard）
  • 少量現金備用，市區匯率比機場好

Connectivity通訊

  • eSIM 預先購買（Holafly / Airalo 支援歐洲多國）
  • Google Maps 離線地圖：布達佩斯 / 尼斯 / 蔚藍海岸 / Verdon
  • 必裝 app：ÖBB、SNCF Connect、DeepL、Google Translate

Packing行李 / 衣著

  • 5–6 月歐洲早晚溫差大，準備薄外套
  • 尼斯是圓石灘 — 涼鞋 / 海灘鞋必備
  • Verdon 峽谷區較涼且海拔高，外套必備
  • 南法陽光強，太陽眼鏡 + 防曬 + 帽子
